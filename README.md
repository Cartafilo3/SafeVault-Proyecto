En este proyecto trabajé para que la aplicación fuera más segura. Primero, mejoré el formulario para que no aceptara datos vacíos. Luego, en la base de datos, creé roles para que no cualquier usuario pueda borrar cosas.

Copilot me ayudó mucho a entender cómo funcionan los ataques de inyección SQL y XSS. Me dio la idea de crear una función que busca símbolos peligrosos (como comillas o etiquetas de script) y los bloquea antes de que dañen el sistema. También me ayudó a escribir las pruebas para estar seguro de que los bloqueos funcionan correctamente.
